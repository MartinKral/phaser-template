# Phaser template 
TypeScript + WebPack and with fullscreen button, link button and preloader.

You can delete .vscode if you do not use VSCode for debugging

### To start:
```
    npm install
```

### To develop:
```
    npm start
```

* At http://localhost:8080/
* Automatic rebuilds


### To build production:
```
    npm run build
```

Then upload index.html + /dist to your webserver.
